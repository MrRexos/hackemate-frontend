# hackemate-frontend

Base frontend para un proyecto de hackathon con Vite, React, TypeScript,
Tailwind CSS y despliegue preparado para Vercel.

## Scripts

- `npm run dev`: inicia el servidor local.
- `npm run build`: comprueba TypeScript y genera `dist`.
- `npm run preview`: sirve la build localmente.
- `npm run lint`: ejecuta ESLint.
- `npm run typecheck`: ejecuta solo la comprobación de TypeScript.

## Estructura inicial

- `src/components`: componentes reutilizables.
- `src/layouts`: estructuras de página.
- `src/pages`: vistas principales.
- `src/data`: datos estáticos o mocks.
- `src/hooks`: hooks reutilizables.
- `src/lib`: configuración y constantes compartidas.
- `src/utils`: utilidades pequeñas.
- `src/styles`: estilos globales y tokens de Tailwind.
- `src/services`: llamadas a APIs y clientes externos.
- `src/types`: tipos compartidos de TypeScript.
- `src/assets`: imágenes y recursos.
